Commit Bot









How to Use the Commit Bot

Follow these steps to use the Commit Bot:

  1. Enter the number of commits you want to generate in the "Number of Commits" field.
  2. Enter the year for which you want the commits to be made in the "Year for Commits" field.
  3. Enter your GitHub username in the "GitHub Username" field.
  4. Generate a GitHub personal access token with the required permissions. See the instructions below for help.
  5. Enter the generated token in the "GitHub Token" field.
  6. Click the "Start Commits" button to begin the process.

Generating a GitHub Token

To generate a GitHub token:

  1. Go to GitHub Settings.
  2. Click on Generate new token.
  3. Select the scopes you need (e.g., repo for full control of private repositories).
  4. Click Generate token.
  5. Copy the generated token and paste it in the "GitHub Token" field.
